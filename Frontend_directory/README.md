# Frontend Structure

## Active app

- `react-app-js` - current production React app (JS + Vite + Bootstrap + SCSS).

## Legacy sources

- `legacy-razmetka-html` - original static HTML mockups.
- `legacy-styles-css` - original CSS files for static mockups.
- `legacy-scripts-js` - old helper scripts for static pages.

## Rule of thumb

- All new frontend work goes only into `react-app-js`.
- Legacy folders are kept only as reference during migration.
